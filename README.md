# Just image Transformers (JiT)

A pytorch implementation of Just image Transformers (JiT) for image generation

### ðŸ’» Environment Configuration

1. **Create a virtual environment using Anaconda Prompt:**
   
   ```bash
   conda create -n jit python=3.10
   ```

2. **Activate the environment:**
   
   ```bash
   conda activate jit
   ```

3. **Install PyTorch and torchvision:**
   
   ```bash
   pip install torch==2.5.1 torchvision==0.20.1 --index-url https://download.pytorch.org/whl/cu124 
   ```

4. **Install remaining packages:**
   
   ```bash
   pip install numpy==1.22 opencv-python==4.11.0.86 timm==0.9.12 tensorboard==2.10.0 scipy==1.9.1 einops==0.8.1 gdown==5.2.0
   ```
